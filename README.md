# MP Game - 河南省全民健身运动大会

全民健身走活动平台，支持线上运动会、虚拟路线等功能。

## 技术栈

- **框架**: Next.js 15.3.2
- **样式**: Tailwind CSS 3.4.17
- **语言**: TypeScript
- **部署**: Vercel

## 功能特性

- 🏃‍♂️ 线上运动会
- 🗺️ 虚拟路线挑战
- 📊 运动数据统计
- 🏆 积分商城
- 📰 体育资讯

## 本地开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```

## 部署

项目已配置自动部署到 Vercel。
